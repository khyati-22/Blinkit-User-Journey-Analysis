# E-commerce Funnel Analysis

## Project Overview

This project analyzes customer behavior across an e-commerce platform using event-level data.

The objective is to understand how users move through the shopping journey and identify where they drop off before making a purchase.

## Funnel Stages

```text
Product View
    ↓
Wishlist
    ↓
Add to Cart
    ↓
Purchase
```

## Dataset Used

The project uses the following datasets:

- users.csv
- products.csv
- orders.csv
- order_items.csv
- reviews.csv
- events.csv

## Tools Used

- SQL
- Python (Pandas, Matplotlib)
- Power BI
- GitHub

## Project Goals

- Analyze user journeys
- Calculate funnel conversion rates
- Identify drop-off points
- Study user behavior
- Generate business recommendations
